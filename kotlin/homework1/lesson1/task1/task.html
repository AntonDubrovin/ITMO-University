Задача 1. Main

Напишите тело функции greet, выводящее приветствие в виде: Hello, NAME!, подставляя вместо NAME переданную в качестве аргумента строку.

Вызовите функцию greet для каждого аргумента командной строки или же, если не передано ни одного аргумента, вызовите greet один раз, чтобы вывести Hello, world!
