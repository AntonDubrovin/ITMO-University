Задача 2c. Fizzbuzz

Реализуйте функцию fizzbuzz, принимающую целочисленный диапазон и выводящую на каждый элемент диапазона по одной строке в консоль. Для чисел в диапазоне, кратных трём, выводите слово fizz, для кратных пяти – слово buzz, а для кратных 15-и (то есть кратных и трём, и пяти) – слово fizzbuzz. Для остальных чисел выводите их строковое представление.

Пример вывода для диапазона 10..15:

buzz
11
fizz
13
14
fizzbuzz
